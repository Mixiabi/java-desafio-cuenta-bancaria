## Windows

Primero abrimos PowerShell y buscamos la ruta del archivo descargado:

                cd D:\ruta\de\tu\pc\src

Ejecutamos el comando " ls -l " ,tienen que aparecer este listado de documentacion:

                Directory: D:\ruta\de\tu\pc\src


                Mode                LastWriteTime         Length Name
                ----                -------------         ------ ----
                d----        20/05/2024     19:59                  com
                -a---        20/05/2024     22:24            350   Desafio.class
                -a---        20/05/2024     19:59            232   Desafio.java

Para ejecutar escribimos:

                javac Desafio.java

Luego para ejecutarla:

                java Desafio

Finalmente se abrira una ventana y puedes comenzar a interactuar
